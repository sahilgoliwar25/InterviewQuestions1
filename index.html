Q1. Difference between Div and Span.

Ans. Div tag is a block element i.e. it uses the whole area and any text added after this tag is automatically shifted to next line. It is generally used to wrap some content which then can be used in designing purpose where as Span tag is an Inline element which utilizes only the area having the content in it. Any other content added after this tag will be continued post this tag. It is generally used to wrap the text or a particular word and highlight make some changes to it.

This is text outside Div tag.

Q2. Difference between class and IDs.

Ans. ID is always unique within an HTML Page whereas Class with the same name can be applied multiple times within an HTML Page. When we use style tag with ID we use id="demo" with tag and #demo for style. Similarly when we use style tag with class we use class="demo" with tag and .demo for style.


Q3. What are self closing tags.

Ans. Self closing tags are those which doesn't need to be closed with a separate closing tag i.e. there is no extra closing tag as </tag>.
Example: br, hr, img, etc. are some of self closing tags. Here we don't write seperate </br> tag. Instead we do <br />.

Q5. Difference between block and inline.

Ans. Block tags are those tags which uses whole width of the screen.
Example: p, div, h1 to h6, etc tags are Block Tags.

Inline tags are those tags which uses only the required width for the content.
Example: img, table, span, lable, input etc tags are Inline Tags.

strong tag is used for changing the text into Bold format. It is used to show Strong importance to text. This tag is similar to b tag.

Q7. What are attributes?

Ans. Attributes are special words which provides the additional information about the element. They are case sensitive and always should be written in lowercase only. We can use multiple attributes in a single tag by seperating them with blank spaces.
Example: <img src="http://example.com" alt="Example"/>
Here src and alt are attributes of img tag.


Q8. Difference between HTML tags and elements.

Ans. HTML Tags- Tags are starting and ending part of HTML element. They begin with less than symbol(<) and ends with greater than symbol(>). Whatever is written inside this symbol are considered as tag.
Example: <p></p> is an example of HTML Tags.
HTML elements- Elements are any content which is enclosed within the tag. The starting tag, content and then the ending tag is general structure of any HTML Element.
Example: <p>Content</p> is an example of HTML Elements.
